# ReceiptWarrantyVault

Native iPhone app for a private, offline-first receipt and warranty vault.

The v1 direction follows `specifiche_global.md`:

- SwiftUI, SwiftData, and Apple-native frameworks.
- English base localization with Italian support.
- No account, no backend, no ads, no analytics, no tracking.
- Paid upfront distribution model, so no StoreKit paywall in v1.
